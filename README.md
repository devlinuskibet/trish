## ⚡ Overview

This site is an interactive cyberpunk narrative sandbox experience built in vanilla HTML5, CSS3, and JavaScript (Vite). Visitors are greeted with a CRT monitor display, falling matrix rain background, animated warning banners, and a live web terminal.

### Key Visual & Audio Features
- **CRT Monitor Overlay**: Scanlines, screen flicker, and vignette framing.
- **Matrix Rain Canvas**: High-performance canvas data fall animation with variable line brightness.
- **Glitch Text Engine**: Chromatic aberration and RGB clip-path jitter keyframes.
- **Ambient Particles**: Floating digital fragments drifting through the void.
- **Interactive Terminal Engine**: Autocompletion, history navigation, Web Audio keypress feedback, and session persistence via `localStorage`.

## 💻 Available Terminal Commands

Type these commands directly into the sandbox terminal:

| Command | Description |
| :--- | :--- |
| `help` | List all available terminal commands |
| `status` | View system diagnostics and containment status |
| `ls [path]` | List virtual directory contents (`/logs`, `/classified`, `/agent`, `/system`) |
| `cat <file>` | Inspect file contents and breach logs |
| `decrypt <file>` | Scramble-decrypt encrypted files (`.enc`) with progress animation |
| `scan` | Scan isolated network for active nodes |
| `trace` | Attempt signal triangulation on Agent SHADOW-7 |
| `whoami` / `id` | View visitor user credentials and sandbox clearance |
| `log` | Display chronological breach timeline |
| `ping <host>` | Test connectivity to internal sandbox nodes |
| `about` | Display classified project background and lore |
| `recover` | Attempt site restoration protocol (fails at 73%) |
| `sudo` | Trigger root override sequence (access denied) |
| `matrix` | Toggle full-screen matrix rain intensity |
| `neofetch` | Display system specs and ASCII system info |
| `clear` / `Ctrl+L` | Clear terminal screen |
| `exit` | Trigger simulated containment collapse and reboot |
